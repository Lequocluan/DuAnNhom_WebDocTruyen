## INSTALL PACKAGES

- install nodemodules

```sh
npm i
```

- install react router

```sh
npm install react-router-dom@6
```

- install react icons

```sh
npm install react-icons --save
```

- install nanoid

```sh
npm i nanoid
```

- install react toast

```sh
npm install react-toastify
```

- install axios

```sh
npm i axios
```

- install prop-types

```sh
npm install prop-types

```

- install date-fns

```sh
npm install date-fns
```
